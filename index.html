Часто задаваемые вопросы

Как установить программу?

1. Клонируйте репозиторий: git clone https://github.com/username/video-downloader.git и перейдите в папку.
2. Установите зависимости: pip install -r requirements.txt
3. Установите FFmpeg (если не установлен)
4. Запустите приложение: python main.py

Как пользоваться программой?

1. Запустите приложение: python main.py
2. Вставьте URL видео в поле "URL видео"
3. Нажмите кнопку "Проверить" для получения доступных разрешений
4. Выберите режим загрузки (видео или аудио)
5. Если выбран режим "Видео", выберите желаемое разрешение
6. Нажмите кнопку "Добавить в очередь"
7. Нажмите кнопку "Загрузить все" для начала загрузки

Какие зависимости нужны для работы программы?

Для работы программы необходимы: Python 3.7 или выше, FFmpeg (для конвертации видео и извлечения аудио), PyQt6, yt-dlp, QtAwesome (для иконок).

Как установить FFmpeg?

Windows: Скачайте и установите с официального сайта
macOS: brew install ffmpeg
Linux: sudo apt install ffmpeg (Ubuntu/Debian) или sudo dnf install ffmpeg (Fedora)
